Application Purpose

The main purpose of the application is to scan a Python dictionary for invalid values. The task is to analyze the dictionary content and check for forbidden characters or values in keys or values.

Built-in Menu Items

File

This menu item is responsible for working with files.

  • Create – create a new file.
  • Open – open an existing file.
  • Save – save the current file.
  • Save As – save the file under a different name.
  • Exit – exit the compiler.

Edit

This menu item contains text editing tools.

  • Undo – undo the last action.
  • Redo – redo the undone action.
  • Cut – cut the selected text and save it to the clipboard.
  • Copy – copy the selected text to the clipboard.
  • Paste – paste text from the clipboard.
  • Delete – delete a text fragment.
  • Select All – select the entire text fragment.

Text

This menu item provides access to theoretical materials.

  • Problem Statement – information about dictionaries, example.
  • Grammar – grammar in Chomsky notation.
  • Grammar Classification – grammar type definition.
  • Analysis Method – state diagram.
  • Test Example – compiler capabilities.
  • References
  • Source Code

Start

This menu item is responsible for starting the compiler. You can also use the button below.

Help

This menu item provides help and information.

  • Help – user guide for the compiler.
  • About – basic program information and contact details.

Localization

This menu item is responsible for switching the language. The most common languages are available: Russian and English.

View

This menu item allows changing the font size.
