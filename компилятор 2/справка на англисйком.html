Application Overview

This application is a compiler with basic file management and text editing functions.

Built-in Menu Items

File

This menu item is responsible for file operations.

  • New - create a new file.
  • Open - open an existing file.
  • Save - save the current file.
  • Save As - save the file under a different name.
  • Exit - exit the compiler.

Edit

This menu item contains text editing tools.

  • Undo - undo the last action.
  • Redo - redo the undone action.
  • Cut - delete the selected text and save it to the clipboard.
  • Copy - copy the selected text to the clipboard.
  • Paste - paste text from the clipboard.
  • Delete - delete a text fragment.
  • Select All - select the entire text fragment.

Help

This menu item allows text formatting.

  • Help - instructions for the compiler.
  • About - main information about the program and contact details.
